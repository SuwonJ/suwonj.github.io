## 정리
- **이산적 vs 연속적**

| 구분 | 이산수학 | 연속수학 |
| :--- | :--- | :--- |
| 영역 | 정수 영역 | 실수 영역 |
| 연속성 | 분리된 원소들 | 연속적인 원소들 |
| 집합 | 가산 집합(유한+무한) | 유한 집합 + 비가산 무한 집합 |

- **수학적 모델링 단계**
	1. 주어진 문제의 상황과 배경 파악
	2. 주어진 문제와 수학적 구조와의 대응
	3. 수학적 기초 개념을 이용한 문제 해결

- **퍼셉트론(Perceptron)**
	- 활성화 함수(Activation Function): $y = f(\sum w_i x_i + b)$
		- Step Function: $f(x) = \begin{cases} 0 & if\ x < 0 \\ 1 & if\ x \ge 0 \end{cases}$
		- Sigmoid: $\sigma(x) = \frac{1}{1+e^{-x}}$
		- ReLU: $f(x) = max(0, x)$
	- 가중치($w$)와 편향($b$): 타겟 함수 $F_{target}$에 대해 $F_{target} \approx f(\sum w_i x_i + b)$가 되도록 하는 값을 찾는 과정이 학습임.
	- 예) NAND 게이트 구현: $w_1 = -1, w_2 = -1, b = 1$일 때 $f(-x_1 - x_2 + 1)$은 NAND 연산 수행.
