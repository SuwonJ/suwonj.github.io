## 범위
- 집합 (p88-)

## 정리
- 집합의 표현
	- 원소 나열법 ${1,2,3,4,5}$ 혹 조건 제시법 $S=\{x|p(x)\}$으로 표현. 
- 집합의 크기(카디날리티, cardinality)
- 수 기호
	- $\mathbb{R}$:실수 집합
	- $\mathbb{Z}$: 정수 집합
	- $\mathbb{N}$: 자연수 집합
	- $\mathbb{Q}$: 유리수 집합
	- $S_n$: 1부터 n까지의 자연수의 집합
- $\subseteq$: 부분집합 (Subset)
- $\subset$: 진부분 집합 (Proper Subset) - $A \subseteq B$ 이지만 $A \ne B$인 경우.
- $A^C$ 또는 $\bar{A}$: A의 여집합.
- 집합의 연산
	- $\cup$ 합집합 (Union): $A \cup B = \{x | x \in A \lor x \in B\}$
	- $\cap$ 교집합 (Intersection): $A \cap B = \{x | x \in A \land x \in B\}$
	- $A-B$ 차집합 (Difference): $A - B = \{x | x \in A \land x \notin B\}$
	- $\oplus$ 대칭 차집합 (Symmetric Difference): $A \oplus B = (A \cup B) - (A \cap B) = (A-B) \cup (B-A)$
	- $A\times B$ ==곱집합 (Cartesian Product)==: $A \times B = \{(x,y) | x \in A \land y \in B\}$ (순서쌍의 집합)

#### 집합의 대수 법칙
- **멱등 법칙**: $A \cup A = A, \quad A \cap A = A$
- **항등 법칙**: $A \cup \emptyset = A, \quad A \cap U = A, \quad A \cup U = U, \quad A \cap \emptyset = \emptyset$
- **보합 법칙**: $A \cup \bar{A} = U, \quad A \cap \bar{A} = \emptyset, \quad \bar{U} = \emptyset, \quad \bar{\emptyset} = U$
- **이중 보법칙**: $\bar{(\bar{A})} = A$
- **교환 법칙**: $A \cup B = B \cup A, \quad A \cap B = B \cap A$
- **결합 법칙**: $(A \cup B) \cup C = A \cup (B \cup C), \quad (A \cap B) \cap C = A \cap (B \cap C)$
- **분배 법칙**: $A \cup (B \cap C) = (A \cup B) \cap (A \cup C), \quad A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **흡수 법칙**: $A \cup (A \cap B) = A, \quad A \cap (A \cup B) = A$
- **드 모르간의 법칙**: $\overline{A \cup B} = \bar{A} \cap \bar{B}, \quad \overline{A \cap B} = \bar{A} \cup \bar{B}$

## 어려운 개념
- 진부분집합기호 부분집합과 섞어 쓰지 않기
- $U=\{1,\{4,5\},4\}$에서 
	- $\{4,5\}\in U$, $\{4,5\}\not\subset U$
	- $\{\{4,5\}\}\subset U$, $\{\{4,5\}\}\not\in U$
	- $\{5\}\not \in U$,$5\not \in U$, $\{5\}\not \subset U$, $5\not \subset U$
	- $\{4\}\not \in U$,$4\in U$, $\{4\} \subset U$, $4\not \subset U$
